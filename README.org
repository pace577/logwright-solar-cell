#+title: Solar Cells and the LogWright function

This Github repository contains code related to the project which led to the paper *"Solar Cells, Lambert W and the LogWright Functions"*. This work is done by Prabhat Lankireddy, Sibibalan Jeevanandam, Aditya Chaudhary under the guidance of Prof. P.C. Deshmukh, Dr. Ken Roberts, and Prof. S. R. Valluri.
